#### Symbolism

|             | .?.+ | .+ |
|:-----------:|:----:|:-: |
| <b>[*]+<\b> | *    | *  |
| <b>/+<\b>   | /    | /  |

Explaination: 
* .?.+ equals 0 or 1 of any character except newline followed by 1 or more of any character except newline
* .+ equals 1 or more of any character except newline
* [*]+ equals 1 or more of the character `*`  
* /+ equals 1 or more `/`
