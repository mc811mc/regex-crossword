# Regex Crossword Solutions with Explanations 

### Tutorial

#### The OR Symbol

|                 | A&#124;B |
|:---------------:|:--------:|
| <b>A&#124;Z</b> | A        |

Explaination: 
* A&#124;Z equals `A` or `Z`
* A&#124;B equals `A` or `B`
